## Run Locally

Clone the project

```bash
  git clone https://github.com/maxtyson123/max-os
```

Go to the project directory

```bash
  cd max-os
```

Make Files and OS

```bash
 make maxos.iso  
```

Run Os 

```bash
     Virtual Box
```
